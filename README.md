
# Desty

A blog as tourist guide.


## Run Locally

Clone the project

```bash
  git clone https://github.com/alxmcr/fe-desty-app-react-ts
```

Go to the project directory

```bash
  cd fe-desty-app-react-ts
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
